HTML

  • HTML is used to create the structure of a webpage
  • An element is a type of content on a webpage, ex.text element, image element, header element, list element
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS - to give the content syle
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to syle a webpage, Inline, Internal, or External CSS style sheet.
  • Inline - When a developer wants to style an element directly in the HTML file
  • Internal - You can embed an entire style sheet directly in the HTML file using style. We link the file usinng "LINK" ELEMENT
  • External- The most common, allows devs to keep all thier CSS rules in a seperate file. Makes the changes easier

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git remote -v to see the connection between local environment and github
  • Git is a version control system that let's you manage and keep track of each versionof your project
  • In the feature branch we make commmit of all the changes we made to the code. Then we push up the changes to our feature branchin our github repo. In order to merge in the main branch,we need to open a pull request

Javascript

    A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script.
  • To Call a function we write the the function name followed by parentenses
  • A function is a set of instructions that tells the computer how to perform a certain task
  • There are 2 requirements for Javascript functions: 1.Define a function 2.Call the function
  • Code block is where we write out the instructions for what the function should do
  • A for loop uses predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • An Array is a single variable that is used to hold a group of data
